## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/ehwilhelmy/max.git
cd max
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:8080](http://localhost:8080) in your browser
